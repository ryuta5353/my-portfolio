# ポートフォリオサイト構築指示書

## 1. プロジェクト概要
現在 `app/page.tsx` のみで構成されている Next.js プロジェクトを、エンジニア志望の大学生にふさわしい、モダンで拡張性の高いポートフォリオサイトへアップグレードしてください。

## 2. デザインコンセプト
* **ベースカラー:** 暗めの紫色（Dark Purple / Deep Violet）を基調としたダークモード。
* **アクセント:** 鮮やかな紫やネオンカラーのグラデーション、微かな光彩（Glow）エフェクト。
* **雰囲気:** 視認性が高く、テック感のある洗練されたスタイル。

## 3. 推奨ディレクトリ構成
以下の構造に従って、ファイルを整理・作成してください。
- `app/`: ルーティングおよび各ページのレイアウト
- `components/ui/`: ヘッダー、フッター、カード、ボタンなどの再利用コンポーネント
- `constants/`: プロフィール、スキル、実績などのテキストデータ（管理を容易にするため）
- `types/`: TypeScriptの型定義
- `public/`: 画像・アセット類

## 4. 実装・改修ステップ
1.  **Tailwind CSS設定:** `tailwind.config.ts` を編集し、独自の暗い紫色（例: `#0a0118` 周辺）を `theme` に追加してください。
2.  **コンポーネント化:** `page.tsx` に直書きせず、以下のセクションをコンポーネントとして切り出してください。
    * **Hero:** キャッチコピーと簡潔な自己紹介。
    * **About:** 芝浦工業大学での学びや、APR（自動プログラム修正）等の研究内容、エンジニアを目指す背景。
    * **Skills:** Next.js, TypeScript, Python などのスキルセット表示。
    * **Projects:** 制作実績をカード形式で表示（ホバー時に紫の光彩が出るなど）。
    * **Contact:** GitHubやSNSへのリンク。
3.  **データ分離:** `constants/index.ts` を作成し、自己紹介文やプロジェクト一覧をオブジェクトとして定義。コンポーネントはそこから `map` 等で描画するようにしてください。
4.  **レスポンシブ:** モバイル端末での表示崩れがないよう、徹底して対応してください。

## 5. 技術スタックの指定
* Next.js (App Router)
* Tailwind CSS
* Lucide React (アイコンライブラリ)
* Framer Motion (もし可能であれば、セクション遷移に滑らかなアニメーションを付与)
